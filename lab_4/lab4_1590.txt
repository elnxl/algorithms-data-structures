Для каждого элемента данной строки (начиная с 1) будем создавать новую строку (с 0 по символ, который рассматривается на данной итерации) и будем рассматривать ее с конца. Затем считаем префикс функцию для созданной строки. Ответом будет: сумма из длин каждой из созданных строк минус наибольший элемент из префикс функции(вычитание выполняется для каждой из созданных строк), которая была посчитана для созданной строки.
